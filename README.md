# DXC-Login-App
1. git clone
```
git clone https://github.com/rkapril/DXC-Login-App.git
```

2. Build docker image
```
docker build -t login-app .
```

3. Run docker image
```
docker run -p 8080:8080 -d login-app
```

## Deployed on:
- https://dxc-login-app.onrender.com
